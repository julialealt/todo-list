## Technologies

React, Typescript and Vite


## Functionalities

- Create a to-do
- Delete a to-do
- Update a to-do status (checked/unchecked)


## Running locally

Clone the project

```bash
  git clone https://github.com/julialealt/todo-list.git
```

Enter the project directory

```bash
  cd todo-list
```

Install the dependencies

```bash
  pnpm i
```

Start the app

```bash
  pnpm run dev
```
